# CAPTCHA Backend

Backend systemu CAPTCHA zbudowany w NestJS. System generuje matematyczne CAPTCHA i weryfikuje ich rozwiązania.

## Wymagania

- Node.js (v14 lub nowszy)
- Redis
- npm lub yarn

## Instalacja

1. Sklonuj repozytorium
2. Zainstaluj zależności:
```bash
npm install
```

## Konfiguracja

Utwórz plik `.env` w głównym katalogu projektu z następującymi zmiennymi:

```env
PORT=3000
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=
```

## Uruchomienie

1. Upewnij się, że Redis jest uruchomiony
2. Uruchom aplikację:
```bash
npm run start:dev
```

Aplikacja będzie dostępna pod adresem `http://localhost:3000`

## API Documentation

Dokumentacja API jest dostępna pod adresem `http://localhost:3000/api` po uruchomieniu aplikacji.

### Endpointy

#### POST /captcha
Generuje nową CAPTCHA.

#### POST /captcha/verify
Weryfikuje rozwiązanie CAPTCHA.

## Testy

Aby uruchomić testy:
```bash
npm run test
```

## Funkcje

- Generowanie matematycznych CAPTCHA
- Weryfikacja rozwiązań
- Ochrona przed brute force (maksymalnie 3 próby)
- Automatyczne wygasanie CAPTCHA po 2 minutach
- Integracja z Redis do przechowywania danych
- Dokumentacja API w Swagger
